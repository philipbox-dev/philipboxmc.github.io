/transform: Set the brush transform

/tree: Tree generator tool

/trigger: A Mojang provided command.

/unban: Unbans the specified player.

/unbanip: Unbans the specified IP address.

/undisguise: Undisguise yourself.

/undisguiseentity: Undisguise an entity.

/undisguiseplayer: Undisguise a player.

/undisguiseradius: Undisguise all entities within a radius.

/undo: Undoes the last action (from history)

/undostatue: Undo the last statue created by ANYONE on the server

/ungod: [player] - Disable godmode on a player

/unlimited: Allows the unlimited placing of items.

/unregister: Unregister your account.

/unstuck: Escape from being stuck inside a block

/up: Go upwards some distance

/vanish: Hide yourself from other players.

/vault-convert: Converts all data in economy1 and dumps it into economy2

/vault-info: Displays information about Vault

/version: Gets the version of this server including any plugins in use

/viaversion: Shows ViaVersion Version and more.

/warning: Sends a warning message to the entire server.

/warp: List all warps or warp to the specified location.

/warpinfo: Finds location information for a specified warp.

/waypoint: Waypoint commands

/weather: Sets the weather.

/whitelist: A Mojang provided command.

/whois: Determine the username behind a nickname.

/workbench: Opens up a workbench.

/worldborder: A Mojang provided command.

/worldedit: WorldEdit commands

/worldguard: WorldGuard commands

/worth: Calculates the worth of items in hand or as specified.
